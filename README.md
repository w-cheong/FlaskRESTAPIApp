

Routes 

* [X] GET `/characters` -- list all characters with pagination
* [ ] GET `/characters/search` -- search for a character by first or last name
* [ ] PUT `/characters/<id>` -- update a character by id and persist to CSV
* [ ] DELETE `/characters/<id>` -- remove a character by id and persist to CSV


Examples:
* `GET /characters/search?first_name=Phoebe`
* `GET /characters/search?last_name=Geller`
* `GET /characters?page=1&per_page=10`
